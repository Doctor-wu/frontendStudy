## 配置多入口的打包编译

把entry设置为一个对象

```javascript
...
entry:{
    index: "./src/main.js",
    login: "./src/login.js"
}
...
output: "[name].[hash].min.js"



const htmlPlugins = ["login", "index"].map(item=>{
    return new HtmlWebpackPlugin({
        template: `./public/${item}.html`,
        filename: `${item}.html`,
        chunks: [item],
        minify: {
            collapseWhitespace: true,
            removeComments: true,
            removeAttributeQutes: true,
            removeEmptyAttributes: true
        }
    })
})



plugins: [
    ...htmlPlugins
]
```
