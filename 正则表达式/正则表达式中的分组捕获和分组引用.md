## 正则表达式中的分组捕获和分组引用



### 正则的分组捕获

```javascript
// => 身份证号
let str = "369200200010014032";
let reg = /^(\d{6})(\d{4})(\d{2})(\d{2})\d{2}(\d)(?:\d|X)$/;// (?:) 只匹配不捕获

// 既要捕获到{数字}，也想单独的把数字也获取到，例如：第一次找到{0}，还要单独获取0
let str = "{0}年{1}月{2}日";
let reg = /\{(\d+)\}/
```



### 分组引用

```javascript
let str = "book";  //=> "good","soon"...
let reg = /^[a-zA-Z]([a-zA-Z])\1[a-zA-Z]$/;  // \1就是匹配和第一个分组一模一样的内容
```
