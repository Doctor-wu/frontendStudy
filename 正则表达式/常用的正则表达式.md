## 常用的正则表达式

### 验证是否为有效数字

```javascript
/**
* 规则分析
* 1. 可能出现 + - 号   [+-]?
* 2. 一位0-9都可以，多位首位不能为0   (\d|([1-9]\d+))
* 3. 小数部分可能有可能没有，一旦有，后面必须有小数点加数字   (\.\d+)?
*/

let validNumReg = /^[+-]?(\d|([1-9]\d+))(\.\d+)?$/
```

### 验证密码

```javascript
// 数字，字母，下划线，6-16位
let reg = /^\w{6,16}$/;
```

### 验证真实姓名

```javascript
// [\u4E00-\u9FA5]

let reg = /^[\u4E00-\u9FA5]{2,10}(\.[\u4E00-\u9FA5]{2,10})*$/
```

### 验证邮箱

```javascript
let reg = /^\w+((-\w+)|(\.\w+))*@[A-Za-z0-9]+((\.|-)[A-Za-z0-9]+)*\.[A-Za-z0-9]+$/
```

### 身份证号

```javascript
/** 
* 1. 一共18位
* 2. 最后一位可能是大写的X
*/

let reg = /^\d{17}(\d|X)$/
```
